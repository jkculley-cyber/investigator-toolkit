## 2. Description of Service

Waypoint is a cloud-based discipline management and DAEP (Disciplinary Alternative Education Program) administration platform designed for Texas K-12 school districts. The Service includes:

- Incident tracking, referral workflows, and placement management
- DAEP orientation scheduling and daily behavior tracking
- Compliance checklists aligned with Texas Education Code requirements
- SPED/Section 504 manifestation determination workflows
- Transition planning and intervention tracking
- Reporting, data export, and alerting features
- Navigator module functionality (ISS/OSS referral and placement tracking), when included in the District's subscription

The Service is hosted on cloud infrastructure (Supabase) and accessed through a web browser. No software installation is required.
